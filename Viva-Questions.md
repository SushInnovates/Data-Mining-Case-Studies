# Data Mining Case Studies – Viva Questions & Answers

---

## 1. What is Data Mining?
Data mining is the process of extracting useful patterns and insights from large datasets using statistical and machine learning techniques.

---

## 2. What is Feature Scaling?
Feature scaling is the process of normalizing data so that all features are on a similar scale. It helps improve model performance.

---

## 3. Why is Feature Scaling important?
Because machine learning algorithms can be biased towards features with larger values. Scaling ensures all features contribute equally.

---

## 4. What is StandardScaler?
StandardScaler transforms data so that it has:
- Mean = 0
- Standard deviation = 1

---

## 5. What is PCA?
PCA (Principal Component Analysis) is a dimensionality reduction technique that converts features into a smaller set of variables while preserving important information.

---

## 6. Why use PCA?
- Reduces number of features
- Improves model performance
- Reduces computation time

---

## 7. What is Classification?
Classification is a machine learning task where the output is a category (e.g., yes/no, 0/1).

Example: Insurance claim approval

---

## 8. What is Regression?
Regression is a machine learning task where the output is a continuous value.

Example: Predicting energy consumption

---

## 9. What is Decision Tree?
A Decision Tree is a model that makes decisions based on conditions and splits data into branches.

---

## 10. What is Random Forest?
Random Forest is an ensemble model that uses multiple decision trees to improve accuracy and reduce overfitting.

---

## 11. What is Accuracy?
Accuracy measures how many predictions are correct out of total predictions.

Formula:
Accuracy = Correct Predictions / Total Predictions

---

## 12. What is MSE?
Mean Squared Error measures the average squared difference between actual and predicted values.

Used in regression problems.

---

## 13. Difference between Accuracy and MSE?

| Metric | Used For |
|------|--------|
| Accuracy | Classification |
| MSE | Regression |

---

## 14. What is Train-Test Split?
It divides the dataset into:
- Training data (to train model)
- Testing data (to evaluate model)

---

## 15. Why preprocessing is important?
Because raw data contains noise, missing values, and inconsistencies. Preprocessing makes data suitable for machine learning models.
