# Data Mining Case Studies

This repository contains a collection of basic data mining and machine learning case studies implemented using Python. The projects focus on real-world datasets and demonstrate core concepts such as classification, feature scaling, and dimensionality reduction.

---

## 📌 Case Studies Included

### 1. Insurance Claim Prediction
- Algorithm: Decision Tree
- Task: Classification
- Objective: Predict whether an insurance claim will be approved or not
- Dataset: Insurance Claims Dataset

---

### 2. Wine Quality Analysis
- Techniques: StandardScaler, PCA
- Algorithm: Random Forest / Logistic Regression
- Task: Classification
- Objective: Predict wine quality based on chemical properties
- Dataset: Wine Quality Dataset (UCI)

---

### 3. Flight Delay Prediction
- Techniques: Data Cleaning, Scaling, PCA
- Algorithm: Decision Tree
- Task: Classification
- Objective: Predict whether flights are delayed
- Dataset: Flight Delay Dataset (USDOT / Kaggle)

---

### 4. Energy Consumption Forecasting
- Techniques: Feature Scaling (StandardScaler)
- Algorithm: Linear Regression
- Task: Regression
- Objective: Analyze impact of scaling on forecasting models
- Dataset: Household Electric Power Consumption (UCI)

---

## 🎯 Key Concepts Covered
